# Argo playground

## How to start argo

1. start minikube

       00-start-minikube

2. deploy argo to minikube

       01-start-argo

3. do port-forward 

4. download `argo` binary to laptop

## How to start workflow

1. Open http://localhost:2746/

2. Submit workflows from `workflow` dir

        argo submit 02-python.yml
